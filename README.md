# Flour Crust Bakery Website

## WEDE5020 Part 1 Assignment

A responsive multi-page website for Flour Crust, an artisan bakery specializing in traditional European baking methods and premium baked goods.

## 🎯 Project Overview

This website showcases a modern, professional bakery website built with HTML5, CSS3, and JavaScript. The site features a warm, bakery-themed design with full responsive functionality across all devices.

## 📱 Live Demo

Open `index.html` in your browser to view the website. The site will automatically redirect to the homepage.
